Les objectifs

L'objectif est de réaliser le prototype interactif d'une application de co-voiturage réservée aux personnels et étudiants de l'université de Franche-Comté. Le nom de l'application est Campus Car.

Date : 2020
Réalisation : DUT MMI
Type : Projet indivuel
